Помощь и секреты

Как играть:

Кликайте по пузырям, чтобы лопать их и набирать очки. Не позволяйте пузырям улететь!

Секретные чит-коды:

В игре есть секретные чит-коды, которые можно вводить во время игры с клавиатуры:

  • bubble - разблокирует радужный режим (нажмите R для активации)
  • pop - разблокирует специальные эффекты (нажмите A для антигравитации)
  • combo - разблокирует супер-комбо (нажмите ПРОБЕЛ для активации)
  • hover - режим наведения (лопайте пузыри без клика, переключайте клавишей H)
  • glass - стеклянные полупрозрачные пузыри (переключайте клавишей L)
  • godmode - активирует все преимущества сразу

Есть и другие секретные коды... Попробуйте их найти!
